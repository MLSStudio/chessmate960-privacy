# ChessMate 960 Privacy Site

This folder contains a static privacy policy page ready for GitHub Pages.

## Quick publish with GitHub Pages

1. Create a new public GitHub repository, for example `chessmate960-privacy`.
2. Upload the contents of this folder so that `index.html` is at the repository root.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` and folder `/ (root)`
5. Save and wait a minute.

Your public URL will look like:

`https://YOUR_GITHUB_USERNAME.github.io/chessmate960-privacy/`

Use that URL in Google Play for the Privacy Policy field.
